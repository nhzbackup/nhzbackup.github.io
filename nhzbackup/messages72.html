~GUIDA INSTALLAZIONE TEMI CUSTOM (CFW)~

Requisiti:
-3DS con CFW Luma (Se avete un 3ds alla 11.5 potete installarlo da http://3ds.guide/it_IT)

1: Scaricate da https://3dsthem.es/ i temi che volete installare, per un massimo di 100
2: Estraete i file zip dei temi in delle rispettive cartelle. Attenzione, dentro le cartelle dei temi non ci devono essere altre cartelle, solo i file dei temi stessi
3: Create una cartella e chiamatela come vi pare. Dentro di essa create una cartella chiamata "Custom" e dentro quest'ultima metteteci le cartelle con i temi che volete installare
4: Aprite l'archivio di Howling Theme Tool, aprite la cartella English e quindi estraete la cartella Software
5: Avviate il programma Theme Checker presente nella cartella Software
6: Aperto il programma, premete "..." e selezionate la cartella creata prima contenente la cartella Custom e quindi i temi. Premete quindi Start Verification.
7: Vi chiederà di inserire i nomi per ogni tema. Se vi va bene quello già scritto, premete semplicemente OK a tutte le caselle di testo che compariranno
8: Una volta finito, avviate Howling Theme Tool e riselezionate la cartella contenente la cartella Custom. I vostri temi appariranno quindi sulla parte superiore dell'applicazione
9: Selezionate i temi uno ad uno e premete Add per farli apparire sulla parte inferiore dello schermo
10: Una volta aver aggiunto ogni tema alla lista di sotto premete (...), quindi Create Package ed infine European (EUR)
11: Selezionate dove salvare il file .CIA e attendete la fine del processo. Una volta finito, passate quel file sulla vostra scheda SD
